# Visual Philosophy

This project follows an editorial premium design language inspired by midu.design.

---

# Core Principles

- Typography-first design
- Spacious layouts
- Minimal color palette
- Smooth motion
- Visual restraint
- Cinematic gradients
- Layered depth
- Clean hierarchy

---

# Typography

Typography is the primary design system.

Typography should:
- create hierarchy
- control rhythm
- create atmosphere

Avoid:
- inconsistent font sizes
- weak spacing
- excessive font weights

---

# Motion

Motion should:
- feel smooth
- feel subtle
- support hierarchy
- support readability

Avoid:
- attention-seeking animation
- chaotic transitions
- inconsistent timing

---

# Color

Color should remain restrained.

The interface should feel:
- dark
- cinematic
- atmospheric
- premium

---

# Visual Density

Prefer:
- whitespace
- clean composition
- breathing room

Avoid:
- clutter
- over-layering
- visual noise

---

# Effects

Effects should:
- enhance atmosphere
- remain soft
- remain intentional

Avoid:
- excessive blur
- excessive glow
- gimmicky visuals

---

# Premium Feel

Premium design comes from:
- spacing rhythm
- typography precision
- motion restraint
- consistency

NOT from excessive effects.
